# ☕ Java School & Personal Projects

Welcome to this repository! Here you'll find my projects, exercises, and workshops developed in **Java** during my school years
and personal study.

The repository is organized into subfolders. **Each project contains its own specific `README.md` file** with detailed instructions
on how to compile and run it.



## 🛠️ Global Prerequisites

To run the projects in this collection, make sure you have the following installed on your computer:

* **Java Development Kit (JDK):** Version 8 or higher (OpenJDK 11+ or 17+ recommended).
* **An IDE of your choice:** Eclipse, IntelliJ IDEA, NetBeans, or simply VS Code.
* **Git:** To clone the repository.
